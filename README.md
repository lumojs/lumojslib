## 🚀 About

Lumo.js is a lightweight JavaScript library focused on creating smooth and interactive user experiences.

It is designed to be:
- Simple to use  
- Performance focused  
- Easily extendable  
- Developer-friendly  

---

## 📦 Installation

```html

```

## 🧪 Usage

```js
lumo.snake({
  color: "#00ff88",
  trailCount: 12,
  magnetic: true,
  hideDefaultCursor: true
});

## 🎯 Philosophy

Lumo.js is built with a simple idea:

> Keep things minimal, fast, and powerful.

Instead of adding unnecessary complexity, it focuses on delivering smooth interactions with clean APIs.
